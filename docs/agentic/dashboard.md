# Agentic Workflow Dashboard

An interactive dashboard for visualizing and analyzing multi-agent system trajectories and CLEAR evaluation results.

## Features

### 1. Workflow View
- Interactive graph visualization of agents and their relationships
- Node statistics (call counts, unique tasks, tool vs. agent calls)
- Edge weights indicating transition frequencies
- Agent selection via dropdown to highlight nodes and view detailed statistics
- Trajectory length distribution and score correlation
- Aggregate scores across dimensions for full traces
- CLEAR analysis summary over all traces

### 2. Node-Specific CLEAR Analysis
- Complete CLEAR analysis results for each agent
- Score distribution visualizations
- Issue frequency table with severity indicators
- Interactive data explorer with drill-down into individual records
- Detailed evaluation view including input, response, evaluation text, and recurring issues

### 3. Trajectory Explorer
- Browse individual trajectories by task ID
- Filter by trajectory length, agents, or score range
- View complete agent flow for each trajectory
- Full-trace evaluation results
- Step-by-step execution details with inputs and outputs
- Score progression visualization across trajectory steps

### 4. Path Analysis
- Identify the most common paths through the workflow
- Path frequency distribution charts
- Success vs. failure pattern analysis
- Dead-end node detection

### 5. Temporal Analysis
- Agent position analysis (where agents appear within trajectories)
- Score progression analysis across trajectory steps
- Agent retry pattern detection and visualization

### 6. Score Prediction
- ROC curve analysis (requires gold trajectory scores)
- Predict trajectory success from average step scores or full-trace scores
- Success rate analysis by trace length

## Installation

The dashboard requires the `clear_eval` package to be installed.

## Requirements

The dashboard requires at least **step-by-step analysis** results to function. Without full trajectory analysis, the dashboard will display partial results—trajectory-level scores and evaluations will be unavailable.

## Usage

### Quick Start

```bash
python -m clear_eval.agentic.dashboard.launch_dashboard
```

### Command-Line Options

```bash
python -m clear_eval.agentic.dashboard.launch_dashboard --port 8080 --host 0.0.0.0 --no-open
```

| Flag | Description | Default |
|------|-------------|---------|
| `--port` | Port to run the dashboard on | `8080` |
| `--host` | Host address to bind to | `0.0.0.0` |
| `--no-open` | Disable automatic browser launch | `False` |

### Getting Started

1. **Launch** the dashboard using the command above
2. **Upload** a `unified_ui_results.zip` file (generated by the CLEAR pipeline) via the sidebar upload widget
3. **Explore** the analysis tabs

## Technology Stack

- **NiceGUI**: Modern Python web UI framework (Quasar/Vue.js under the hood)
- **Plotly**: Interactive chart visualizations
- **NetworkX**: Graph analysis and layout algorithms
- **Pandas**: Data processing and manipulation

## Tips

1. **Performance**: Initial load may take a moment for large datasets. Data is processed once on upload.

2. **Graph Visualization**:
   - Larger nodes indicate more calls
   - Thicker edges indicate more transitions
   - Orange nodes are selected; blue nodes are unselected

3. **Filtering**: Use the Advanced Filters panel in Trajectory Explorer to narrow results by length, agent, or score range.

4. **Data Explorer**: Click any row in the Node Analysis data table to view full record details.
